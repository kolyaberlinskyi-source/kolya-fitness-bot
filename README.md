# KolyaFitness Bot 🤖💪

Персональный фитнес-коуч в Telegram с анализом данных Oura Ring через Claude AI.

## Команды бота
- `/start` — приветствие
- `/today` — анализ сегодня
- `/week` — анализ за неделю  
- `/tip` — совет на сегодня
- Любое сообщение — умный ответ агента

## Деплой на Railway

1. Зайди на railway.app
2. New Project → Deploy from GitHub repo
3. Загрузи эти файлы в GitHub репозиторий
4. Railway сам найдёт Procfile и запустит бота

## Файлы
- `bot.py` — основной код
- `requirements.txt` — зависимости
- `Procfile` — команда запуска для Railway
